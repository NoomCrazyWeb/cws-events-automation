## Description

The **CWS Events Automation** plugin retrieves Ninja Forms submissions and posts them as events with a Draft status. Perfect for websites that require event management based on form submissions, this plugin simplifies the process by automating the submission-to-event flow.

### Features:
- Automatically converts Ninja Forms submissions to draft events.
- Seamless integration with Ninja Forms and event post types.
- Simple and easy-to-configure settings.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/cws-events-automation` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin via **Dashboard** -> **CWS Ninja Forms Submissions Events**.

### Manual Installation
1. Download the plugin’s .zip file.
2. Go to your WordPress dashboard and click on "Plugins" > "Add New" > "Upload Plugin".
3. Upload the .zip file and click "Install Now".
4. Once installed, activate the plugin.

## Frequently Asked Questions

### Q1: How do I configure the plugin?

A: After activation, go to **Dashboard** -> **CWS Ninja Forms Submissions Events** to configure how the submissions are retrieved and converted to events.

### Q2: Can I customize the event fields?

A: The plugin uses standard event post types and fields, but customization is possible through additional WordPress hooks and filters.
